marley
=====

An OTP application

Build
-----

    $ rebar3 compile

# marley

## Description

OTP application

Very basic webserver in erlang.

## Usage
```bash
# build
$ rebar3 compile

# remove temporary files
$ rebar3 clean

# run unit tests
$ rebar3 eunit

# run system and integration tests
$ rebar3 ct

# run property tests
$ rebar3 proper

# run linter
$ rebar3 lint

# Runs cross reference analysis.
$ rebar3 xref

# Static code analysis
$ rebar3 dialyzer

# Generate documentation with edoc
$ rebar3 edoc

# Start shell with application loaded
$ rebar3 shell

# Start shell with application loaded and listen for code changes
$ rebar3 auto

# Run release
$ rebar3 run

```
